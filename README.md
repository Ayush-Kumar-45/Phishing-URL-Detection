# Machine Learning Based Phishing URL Detection System

A web-based application that uses machine learning to detect phishing websites by analyzing suspicious patterns in URLs.

## Features

- 🔍 **Real-time URL Analysis**: Enter any URL and get instant results
- 🤖 **Machine Learning Model**: Random Forest classifier with 96% accuracy
- 📊 **Feature Extraction**: Analyzes 35+ URL characteristics
- 🎨 **User-friendly Interface**: Clean, responsive design
- 📜 **Search History**: Track your URL checks
- 📱 **Mobile Responsive**: Works on all devices

## System Architecture

The system follows a three-tier architecture:
- **Presentation Layer**: HTML, CSS, JavaScript frontend
- **Application Layer**: Flask backend
- **ML Detection Engine**: Scikit-learn Random Forest model
